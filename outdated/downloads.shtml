ProteoWizard Downloads

There are several flavours of ProteoWizard depending upon your intended use. If you are primarily interested in using the tools, the binary distributions for your platform are most likely what you want. If you are interested in using ProteoWizard as part of a software project you are developing, you should get the source distribution, or download from our subversion repository directly.

Though we are formally at version 2.1, we use a continuous integration environment (each time a developer checks in an update, ProteoWizard is built from scratch, run through a battery of tests, and then published to our website for download). If you would like an older version, they are archived on our TeamCity site.

For binaries, there is no installation process: the executables are mostly command-line interfaces that run wherever they are extracted. For help with this see our user documentation.
