Building ProteoWizard for use in your own Windows programs

Check the downloads page for information on downloading source and binaries for use in your own projects.

Officially, the ProteoWizard libraries and tools are built using the bjam build system, but we also provide alternate semi-official build files for MSVC (.sln and .vc(x)proj for VC8, 9, and 10) and for GNU/Autotools (the familiar old "configure ; make ; make install"). Note that these alternate build files do not support vendor DLLs: maintaining that capability is complex and thus limited to the officially supported bjam build. You can read more about it here.

If you download the files for building with MSVC or GNU/GCC, you will find many examples of programs that link with the ProteoWizard library.
